# 主要页面原型说明（用于AI原型生成）

> 建议使用 Visily（中文友好，支持文本生成线框）。如用英文工具（Uizard/Galileo/Figma AI），可将提示翻译为英文再生成。通用风格提示：`企业后台 Web，浅色、卡片化、中文界面，突出安全与审核，发布即固化。`

## 通用布局
- 顶部：全局导航（工作台、功能市场、Skill、功能开发、权限与审核、运行监控、成本统计、个人中心）+ 搜索框。
- 左侧：项目/部门筛选、标签筛选、最近使用/收藏。
- 右侧浮层：通知、执行状态/任务进度、帮助。
- 重要提示：无权限按钮灰显并给出提示；显著展示“零原文件存储，处理后即清理”。

## 页面说明与生成提示

### 1) 登录 / 注册 / SSO
- 布局：居中卡片。
- 字段：邮箱/手机号、密码；按钮：登录、注册、企业 SSO；复选：记住我；链接：忘记密码。
- 安全：2FA 开关占位；底部提示“零原文件存储，处理后即清理”。
- 提示示例：`企业后台风格登录页，居中卡片，字段邮箱/手机号+密码，按钮登录/注册/企业SSO，复选记住我，忘记密码链接，底部安全提示。`

### 2) 工作台首页
- 卡片三列：最近使用（功能名/语言/上次调用时间）、收藏/常用（标签、调用按钮）、待办审核（功能发布/修改/下架、Skill 变更，状态标签）。
- 推荐/热门功能卡片网格：显示运行时徽标、调用量、成本级别。
- 快捷入口：上传文档、创建 Skill、创建功能、查看统计。
- 顶部搜索框，全局导航。
- 提示示例：`工作台三列卡片：最近使用、收藏/常用、待办审核；下方推荐功能卡片网格（含运行时徽标、调用量、成本标签）；顶部搜索框+导航。`

### 3) 文档上传与处理（零原文件存储）
- 上传组件：拖拽/按钮 + URL 输入（云链接）。
- 支持格式提示：PDF/Word/Excel/图片(OCR)/PPT/CSV，大小/批量限制。
- 步骤进度：解析 → 信息提取 → Skill 生成 → 清理临时数据。
- 结果预览：提取字段/规则，手动标注/修订开关。
- CTA：生成 Skill、丢弃（立即清理）。
- 安全提示：不保存原文件。
- 提示示例：`上传页，左大框拖拽/按钮，URL 输入；右侧步骤进度；下方预览提取字段+标注；按钮生成Skill/丢弃；醒目“不存储原文件”。`

### 4) Skill 管理
- 表格：名称、版本、标签、来源文档、创建者、更新时间、状态（草稿/已发布）、被调用次数。
- 筛选：项目/部门、标签、状态、时间、创建者。
- 详情抽屉：知识点/规则列表，版本切换，下方引用关系（被哪些功能使用），审计日志。
- 操作：新版本、标签管理、分享范围；无权限按钮灰显+提示。
- 提示示例：`Skill 管理表格含名称/版本/标签/来源/创建者/更新时间/状态/调用次数；顶部筛选；点行开右侧抽屉：知识点列表、版本选择、引用关系、审计日志；按钮新版本/标签/分享。`

### 5) 功能开发向导
- 顶部步骤条：来源（模板/Skill/空白）→ 基础信息 → 运行时 → 输入输出 → 依赖 → 测试 → 发布。
- 基础信息：名称、描述、标签、项目/部门。
- 运行时：单选 Python / Node.js / Java（显示配额、冷启动提示）。
- 输入输出：JSON Schema 表单 + 示例输入/输出。
- 依赖：上传/编辑 requirements.txt 或 package.json；Java 选 pom.xml。
- 代码区：高亮编辑器（仅草稿可改）。
- 测试：输入示例，同步/异步运行，显示结果/日志/耗时/成本估算。
- 发布：提示“发布即固化，修改需新版本+审核”，提交审核按钮。
- 提示示例：`功能开发向导页，步骤条；表单含名称/描述/标签/项目；运行时单选Python/Node/Java并显示配额与冷启动；IO定义JSON表单；依赖上传；代码编辑器；测试区运行示例并显示日志/耗时/成本；发布提示固化+审核。`

### 6) 功能管理（固化与版本）
- 表格：功能名、版本、状态（草稿/已发布/下架申请/审核中）、运行时、调用量、P95 耗时、月成本、共享范围、是否市场上架。
- 筛选：状态、运行时、项目/部门、标签、成本级别。
- 详情抽屉/页：
  - 基础信息、运行时、依赖摘要、输入输出定义。
  - 版本历史时间线；可查看旧版。
  - 审核记录（发布/修改/下架）。
  - 调用示例：REST/WebSocket/SSE（curl/JS/Python 片段）。
  - 小图表：近 30 天调用量、成功率、耗时 P95。
- 操作：新版本、申请修改、申请下架、调整共享（需权限+审核）；无权限灰显。
- 提示示例：`功能管理表格含状态/运行时/调用量/P95/成本/共享/市场；筛选；详情抽屉含版本历史、审核记录、调用示例、统计小图；操作含新版本、申请修改/下架、调整共享。`

### 7) 功能市场
- 筛选/排序：用途（摘要/提取/转换/Q&A/审核）、运行时语言、标签、部门/项目；排序热门/评分/低成本。
- 卡片：名称、简介、运行时徽标、评分、调用量、成本级别、共享范围。
- 详情：同功能详情，突出调用示例和成本/性能。
- 提示示例：`功能市场卡片网格，显示名称/简介/运行时徽标/评分/调用量/成本/共享范围；顶部筛选用途/语言/标签/部门，排序热门/评分/低成本；详情页含调用示例和成本性能信息。`

### 8) 权限与审核
- 待办列表：类型（功能发布/修改/下架、Skill 变更）、对象名、申请人、时间、状态。
- 审核抽屉：差异对比（代码或配置 diff）、风险提示（运行时/依赖/成本变化），审批按钮（通过/拒绝，备注必填）。
- 权限矩阵展示：角色 vs 操作。
- 审计日志：过滤与导出。
- 提示示例：`审核中心表格列类型/对象/申请人/时间/状态；点行开抽屉显示diff、风险提示、审批按钮（通过/拒绝备注）；下方权限矩阵与审计日志列表。`

### 9) 运行与监控
- 运行时池看板：Python/Node/Java 容量、预热数、健康状态、冷启动耗时。
- 任务队列：排队/执行中/失败重试。
- 图表：耗时分布、P95、错误率，按功能/语言切换。
- 告警列表：资源不足、失败率高。
- 提示示例：`运行监控页，上方卡片显示各语言容器池容量/预热/健康/冷启动；中部任务队列表；下方图表耗时/错误率按功能或语言切换；右侧告警列表。`

### 10) 成本与统计
- 成本看板：模型/功能/团队分解；配额进度条 80/90/100% 预警。
- 使用统计：调用量、成功率、耗时趋势，维度切换（项目/部门/功能）。
- 成本优化提示：低成本模型建议、缓存命中率提示。
- 提示示例：`成本页，顶部配额进度条与预警；左侧饼/条形图按模型/功能/团队分解成本；右侧折线图显示调用量/成功率/耗时趋势，可切换维度；底部列出成本优化建议。`

### 11) 个人中心 / 密钥管理
- API 密钥列表：企业密钥（只读）、个人密钥（新建/轮换），最近使用时间。
- 偏好：默认模型、默认运行时语言、主题。
- 安全：2FA 开关、最近登录记录。
- 提示示例：`个人中心卡片：API密钥列表（企业只读、个人可新建/轮换，展示最近使用）；偏好：默认模型/运行时/主题；安全：2FA开关、最近登录记录。`

## 使用说明（生成原型）
1. 打开 Visily，选择“从文本生成”，粘贴对应页面的提示文本。  
2. 统一风格：企业后台、浅色、卡片化、中文；无权限按钮灰显、强调“发布即固化”与“零原文件存储”。  
3. 生成后检查：权限提示、成本/安全提示、发布固化警示、调用示例展示。  
4. 若需英文，保持结构不变，翻译提示文本再生成。
